## Descripción del Bot
Este bot fue desarrollado en Python utilizando la librería `pyTelegramBotAPI`. Su función principal es responder a comandos específicos para demostrar la comunicación bidireccional entre usuarios de Telegram y un servidor backend.

## Lista de Comandos
El bot responde a los siguientes comandos:
* `/hola` - Devuelve un mensaje de saludo.
* `/hora` - Consulta y devuelve la hora actual del servidor.
* `/contacto` - Muestra la información de contacto del equipo.
* `/proyecto` - Da una breve descripción de nuestro proyecto final del curso.
